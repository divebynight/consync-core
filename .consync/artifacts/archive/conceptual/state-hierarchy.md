# State Hierarchy (v1)

## Filesystem
The filesystem is the source of truth for the current system.

## Execution State
`.consync/state/handoff.md` holds the latest execution state for the current workflow.

## Historical State
Future historical state may be recorded in `.consync/artifacts/03_work-log.md`.

## Planning State
Future planning state may be recorded in `.consync/artifacts/archive/system/feature-map.md`.

## Note
Any non-filesystem layer is derived, replaceable, or reconstructable.
